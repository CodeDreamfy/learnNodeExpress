# learn Express

### 启动
```shell
supervisor --w index
```


### 目录结构
>models: 存放操作数据库的文件
public: 存放静态文件，如样式、图片等
routes: 存放路由文件
views: 存放模板文件
index.js: 程序主文件
package.json: 存储项目名、描述、作者、依赖等等信息

### 模板引擎

[nunjucks](https://mozilla.github.io/nunjucks/api.html)
